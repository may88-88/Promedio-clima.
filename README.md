# Comparación entre Programación Tradicional y Programación Orientada a Objetos

## Programación Tradicional
En la programación tradicional, el programa se desarrolla de forma
procedimental, utilizando funciones para realizar tareas específicas.
En este proyecto se usaron funciones para ingresar las temperaturas
diarias y calcular el promedio semanal del clima.

Este enfoque es más sencillo y adecuado para programas pequeños,
pero puede ser más difícil de mantener cuando el programa crece.

## Programación Orientada a Objetos (POO)
La programación orientada a objetos organiza el código mediante clases
y objetos. En este proyecto se creó una clase que encapsula las
temperaturas y utiliza métodos para ingresar los datos y calcular
el promedio semanal.

La POO ofrece ventajas como mejor organización, reutilización del código
y mayor facilidad de mantenimiento.
